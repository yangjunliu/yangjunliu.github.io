5个常用的词根

介绍5个常用的词根，会介绍他们的来源，意思，读音以及例子。帮组我们记忆英语单词。

No.1 scribe / script

1. 来源

  • 来自拉丁语动词 scribere，意思是 “to write”（写）。
  • 原始拉丁拼写：scribere（发音大约为 /ˈskriː.be.re/）

2. 词根含义

  • scribe 的基本含义是：“写”。
  • 它与“文字、记录、描述”有关。
  • 变体词根还有 script（名词形式，意为“书写的东西”）

3. 读音

  • scribe：/skraɪb/
  • script：/skrɪpt/

4. 例词解析

单词 构成 中文含义
describe de（向下）+ scribe（写） 描述
prescribe pre（在前）+ scribe（写） 开药方；规定
subscribe sub（在下面）+ scribe 订阅，签署
transcribe trans（转移）+ scribe 抄写，转录
inscribe in（里面）+ s

5. 记忆小技巧

  • scribe = 写：想象成“scribe 用羽毛笔在羊皮纸上书写”
  • script = 写下的东西：如电影剧本、医生开的处方、附言（P.S.）

No.2 norm

1. 来源

  • 自拉丁语 norma，意为：木匠用来测直的“角尺”，引申为“准则、规范”
  • 后来演变为现代英语中表达“规则”“正常”“标准”的一系列词汇。

2. 词根含义

  • 原始含义 “直尺；标准；规则”

3. 发音

norm：/nɔːrm/

4. 例词解析

单词 构成结构 中文含义
norm norm 常规；标准；规范
normal norm（标准）+ -al（形容词后缀） 正常的，标准的
abnormal ab（离开）+ norm（标准） 反常的，异常的
enormous e（出）+ norm（规则） 巨大的（超出常规）
normalize norm（标准）+ -ize（使…化） 正常化，使标准化
subnormal sub（低于）+ norm（标准） 低于正常的

5. 记忆小技巧

  • norm = standard（标准）想象成一把“尺子”，它衡量什么是“正常”。
  • abnormal = 离开标准的 → 异常的

No.3 view

1. 来源

  • 来自拉丁语 videre，意思是“to see（看见）”
  • 词根变形有：
    • vid（如 video）
    • vis（如 visual）
    • view（法语借词形式）

2. 词根含义

  • 原始含义 “看、看见”
  • 英文含义 see, look, observe（看）

3. 发音

  • view：/vjuː/

4. 例词解析

单词 构成结构 中文含义
view view 视野；看法
review re（再次）+ view 复习，回顾；评论
preview pre（在前）+ view 预览，预告片
interview inter（互相）+ view 面试，访谈
overview over（全面）+ view 概览，概况
supervise super（在上）+ vis（看） 监督（= 从上面看）
vision vis（看）+ -ion（名词） 视力，视野，想象力
television tele（远）+ vis（看） 电视（远距离观看）

5. 记忆小技巧

view = 看：你看到的就是你的 view（视野、观点）
review = 再次看 → 复习、评论
interview = 互相看 → 面试时彼此观察了解

No.4 press

1. 来源

  • 来自拉丁语 premere，意为 “to press down”（向下压）
  • 后来进入古法语为 presser，再传入英语
  • 英语中的 press 既可以做动词（按压），也能做名词（印刷、新闻界）

2. 含义

原始含义“压，挤”

3. 发音

press：/pres/

4. 常见单词例子

单词 构成结构 中文含义
press press 按，压；印刷；新闻界
pressure press + -ure（名词后缀） 压力
express ex（向外）+ press 表达；快递
impress im（向内）+ press 使印象深刻
suppress sub（在下）+ press 镇压，压制
oppress ob（反）+ press 压迫，压制
depress de（向下）+ press 使沮丧；经济下行
compress com（共同）+ press 压缩
repress re（回）+ press 抑制，压抑

5. 记忆小技巧

  • press = 压：想象一个人用手去“压按钮”
  • express = 往外压出 → 表达、表现
  • suppress = 向下压 → 镇压
  • depress = 向下压情绪 → 沮丧

No.5 serve

1. 来源

  • 来自拉丁语 servire：意为“to be a servant”（做仆人）
  • 由名词 servus（奴仆）演变而来
  • 在英语中保留了“服务、服侍、为他人工作”的核心含义

2. 含义

原始含义“做仆人，服务”

3. 发音

serve：/sɜːrv/

4. 常见单词例子

单词 构成结构 中文含义
serve serve 服务；端上；服役
service serve + -ice（名词后缀） 服务；服务行业
servant serve + -ant（人） 仆人
deserve de（加强）+ serve 值得；应得
preserve pre（提前）+ serve 保护；保存
conserve con（共同）+ serve 节约，保存
observe ob（朝向）+ serve 观察
reserve re（回）+ serve 保留；预订
subservient sub（在下面）+ serve + -ient 屈从的；卑躬屈膝的

5. 记忆小技巧

  • serve = 服务：想象一个服务员在端盘子
  • reserve = 服务回来 → 保留、预订
  • deserve = 服务值得 → 应得
  • preserve = 提前服务 → 保存（保护）
